# **App Name**: ReelCraft Portfolio

## Core Features:

- Resume Parsing: Automatically extract key information from the provided resume to pre-fill portfolio sections.
- Video Showcase: Upload and display video editing projects, reels, and YouTube content in a visually appealing format.
- Project Details: Add descriptions, skills used, and links to projects (e.g., YouTube, Instagram) for each showcased video.
- Category Filters: Filter projects by type (e.g., Short-form Content, Long-form Content, Reels, YouTube Videos).
- Contact Form: Enable potential clients to contact Sourabh directly through the portfolio site.

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2) to convey trust and creativity.
- Background color: Light blue (#E1F5FE), desaturated to 20% to create a clean, modern backdrop.
- Accent color: Purple (#9C27B0), chosen for its contrast to the primary blue and its association with creativity.
- Font for headlines: 'Poppins', a geometric sans-serif, to provide a precise and fashionable look for headlines.
- Font for body text: 'PT Sans', a humanist sans-serif, will pair with Poppins to create a modern look with a little warmth for longer blocks of text.
- Use simple, modern icons to represent project categories and skills.
- Employ a grid-based layout for a clean and organized presentation of projects.
- Use subtle transitions and animations to enhance the user experience.
